dentro un file html ci presenta diversi strumenti di manipolazioni con la quale presentare le nostre informazioni, ma prima di adentrarci introduciamo alcuni concetti fondamentali come ad esempio...

Concetto di un elemento

un elemento di html è considerato tutto ciò che è compreso dall'inizio di un tag fino alla sua chiusura.
buona parte delle volte gli elementi saranno "annidate"

Stuttura di un documento html

un documento html è composto da 3 principali aree :
  • il tag "html" ( ovvero la box principale nella quale lavoriamo ).
  • il tag "head" (dove dichiariamo informazioni innerente al documento html).
  • il tag "body" (dove il contenuto viene inserito).
